18. ΔABC এ cosA=sinB-cosC হলে ত্রিভূজটি হবে-

KUET 05-06,RUET 08-09,RU F 13-14,RU C 14-15

এই সমীকরণটি প্রমাণ করতে আমরা ট্রাইগোনোমিতি এবং ত্রিভূজের কোণের সম্পর্কের ধারণা ব্যবহার করবো।

প্রমাণ:

  1. সমীকরণ পুনর্বিন্যাস: প্রথমে সমীকরণটি cosA এর জন্য সমাধান করি: cosA = sinB - cosC cosA + cosC = sinB

  2. ত্রিভূজের কোণের সম্পর্ক ব্যবহার: এই সমীকরণটিতে, A, B এবং C হলো ΔABC ত্রিভূজের কোণ। ত্রিভূজের কোণের সম্পর্ক অনুসারে, আমরা জানি যে: A + B + C = 180°

    এই সম্পর্কটি সমীকরণে প্রতিস্থাপন করলে পাই: cosA + cosC = sinB A + B + C - B = sinB A + C = sinB + B

  3. ত্রিভূজের কোণের সাইন ফাংশন ব্যবহার: এই সমীকরণটিতে, sinB এবং sinC কে ত্রিভূজের বাহুর দৈর্ঘ্যের সাথে সম্পর্কিত করা যায়: sinB = BC / AB sinC = AC / AB

    এই সম্পর্কগুলো সমীকরণে প্রতিস্থাপন করলে পাই: A + C = BC / AB + B A + C = (BC + AB) / AB

  4. ত্রিভূজের অসমতা ব্যবহার: ত্রিভূজের অসমতা অনুসারে, আমরা জানি যে: AB < AC + BC

    এই অসমতা সমীকরণে প্রতিস্থাপন করলে পাই: A + C = (BC + AB) / AB A + C < 2

  5. সমীকরণ সমাধান: A এবং C এর জন্য সমাধান করলে পাই: A < 2 - C C < 2 - A

    এই দুটি সমীকরণকে একত্রিত করলে পাই: 0 < A + C < 4

    এই সমীকরণটি 180° = A + B + C সম্পর্কের সাথে সামঞ্জস্যপূর্ণ।

  6. সিদ্ধান্ত: এই সমীকরণ থেকে, আমরা দেখতে পাচ্ছি যে A এবং C এর মান 0° থেকে 90° এর মধ্যে হতে হবে।

    যদি A বা C এর মান 0° হয়, তাহলে ত্রিভূজটি সমান্তরাল বাহুযুক্ত হবে, যা সমকোণী ত্রিভূজের একটি বিশেষ ক্ষেত্রে।

    যদি A এবং C উভয়ের মানই 0° না হয়, তাহলে ত্রিভূজটি অবশ্যই সমকোণী হতে হবে কারণ A + C এর মান 90° এর চেয়ে কম হতে পারে না।

উপসংহার:

ΔABC ত্রিভূজে cosA=sinB-cosC হলে ত্রিভূজটি সমকোণী হবে।

এই প্রমাণটি ট্রাইগোনোমিতি এবং ত্রিভূজের কোণের সম্পর্কের ধারণা ব্যবহার করে দেখানো হয়েছে যে সমীকরণটি কেবলমাত্র তখনই সন্তুষ্ট হয় যখন ত্রিভূজটি সমকোণী হয়।
